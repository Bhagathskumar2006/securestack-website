// 01
Project Overview
🌐

Static Website

A responsive HTML/CSS website hosted on AWS S3 as a static site, served globally via CloudFront CDN with HTTPS enabled.

☁️

AWS Deployment

Deployed on Amazon S3 with a public bucket policy scoped only to CloudFront. No direct S3 access allowed from the internet.

🔒

Security First

HTTPS enforced, HTTP redirected, Content Security Policy headers, S3 Block Public Access, and CloudFront Origin Access Control configured.

// 02
Security Measures Implemented
Security Measure Tool/Service Purpose Status
HTTPS Enforcement AWS CloudFront All traffic encrypted in transit; HTTP auto-redirects to HTTPS ACTIVE
Origin Access Control S3 + CloudFront OAC S3 bucket is private; only CloudFront can read it ACTIVE
Block Public Access AWS S3 Prevents accidental exposure of S3 objects to the public ACTIVE
Content Security Policy HTML Meta Tag Prevents XSS attacks by restricting resource loading origins ACTIVE
X-Frame-Options: DENY HTML Meta Tag Prevents clickjacking by disabling iframe embedding ACTIVE
X-Content-Type-Options HTML Meta Tag Prevents MIME-type sniffing attacks ACTIVE
IAM Least Privilege AWS IAM Only required S3 permissions granted; no admin-level access CONFIGURED
Bucket Policy AWS S3 Restricts GetObject to CloudFront distribution only ACTIVE
